    NFT的高级玩法

    NFT代表了数字内容的未来。它们是专为数字时代打造的链上、可编程、可交易、无需许可的 DIY 媒体乐高积木。

    这些空白的数字画布改变了各行各业的创作者的游戏规则。但谈论和盯着它们是一回事，而开始尝试在你的投资组合中直接暴露 NFT 是另一回事。那么你从哪里开始呢？

    这就是许多人最近发现自己的地方，无论是加密新手还是老手。无论您来自哪个行业，您的游戏都不会感到羞耻：现在每个人都在同一条船上，试图抓住 NFT 及其潜力，从小艺术家到大机构。

    那么，人们如何开始在蓬勃发展的 NFT 经济中占据有前途的职位呢？

    这是一个与以往一样中肯的问题，看看像Mark Cuban和Chamath Palihapitiya这样的亿万富翁

    投资者目前是如何一头扎进NFT 的。这场千载难逢的文化派对，你想迟到吗？

    该策略将向您展示如何通过多种方式构建您的 NFT 投资组合，包括直接购买 NFT、购买 NFT 平台代币或购买 NFT 指数基金代币。

    当然，一定要自己研究！

    • 目标：在您的加密投资组合中获得 NFT 风险

    • 技能：简单/中级。

    • 努力： 15-30 分钟即可开始。

    • 投资回报率：随着 NFT 市场继续成为主流，如果您多年来一直使用 NFT，这可能很重要。

    如何进入 NFT 的世界

    有多种方法可以接触 NFT，所以幸运的是，您有多种选择。最终，由您决定哪些方法（如果有）适合您的投资组合。

    一个简单的起点是从您认为有吸引力的项目和创作者那里购买您价格范围内的 NFT。您将以与管理 ERC-20 代币相同的方式来保管这些 NFT，例如通过APP钱包或浏览器钱包（如麦子钱包）。

    但是还有其他有趣的方法可以利用 NFT 生态系统的优势。

    我将在这里分解您的所有主要选项，以便您可以开始设计您的个人 NFT 策略。

    直接拥有 NFT

    以太坊已经拥有多个通用 NFT 市场，例如OpenSea和Rarible，它们聚合了来自整个生态系统的 NFT，从而让用户可以通过单一的、用户友好的仪表板购买各种 NFT。

    来源：opensea.io

    此外，还有专门的 NFT 市场，如SuperRare和Nifty Gateway，它们专门以加密艺术 NFT 为中心。

    当然，你可以使用这两种平台，但在你做任何事情之前，你需要留出一些 ETH 来购买和/或竞标 NFT。

    OpenSea 等市场允许以多种代币进行买卖，但 ETH 通常是首选的主要货币。你还需要一个加密钱包，这样你就可以直接与 NFT 市场的智能合约进行交互。

    来源：superare.co/discover

    一旦您准备好加密货币和钱包，就可以去购物了。前往您选择的市场并连接您的钱包，您就可以开始收藏了。

    制定一个预算，然后寻找价格范围内的 NFT。您可能无法负担得起所有吸引您眼球的酷炫 NFT，但肯定有各种价格、各种背景的创作者提供的 NFT。

    当然，构建 NFT 集合的方式不一定是正确或错误的。只需意识到 NFT 是非流动资产，并且您不能保证收回您花费的 ETH。请记住：价值是主观的——一切都在旁观者的眼中！

    现在不乏利用 NFT 的机会或市场效率低下的情况。

    但是，您最安全的方法是购买您个人认为非常酷或独特且您不介意拥有多年的 NFT。潜在的好处是，其他人最终可能也会发现你的 NFT 也很酷，并在未来的某个时候提供有利可图的报价。

    基于 NFT 的 ERC20 代币

    以太坊 ERC20 代币可用于代币化或代表任何事物，因此我们很自然地看到这些代币部署在 NFT 生态系统中的一系列工具用例中。

    例如，像Rarible 的 RARI或Axie Infinity 的 AXS这样的治理代币。这些项目以 NFT 为重点，因此很可能受益于 NFT 的繁荣。

    这些代币允许持有者自下而上地管理这些项目，并将它们作为社区共同拥有，而不是由公司所有、自上而下和采掘式的。

    如果这些 NFT 项目从这里开始大幅增长，对其治理代币的需求应该会相应增长。在 Uniswap 等 DEX 上购买这些代币是接触 NFT 生态系统的一种方式，尽管这些特定资产也可以通过生态系统参与获得（例如在 Rarible 上进行买卖或定期玩 Axie infinity）。

    来源：axie.substack.com

    接下来，还有实用型代币，如Meme Protocol 的 MEME和Sandbox 游戏平台的 SAND 代币。通过实用代币，我的意思是这些资产为其持有者提供了一些不持有它们就无法获得的优势或效用。

    例如，Meme Protocol 是一个 NFT 农业项目，需要用户质押 MEME 才能参与其标志性的 NFT 掉落，而 SAND 代币是用于支付 The Sandbox 世界中游戏内体验的本地货币。

    这些资产可以捕捉对这些 NFT 项目不断增长的需求，因此它们是另一个需要考虑的途径。

    最后一类要考虑的以 NFT 为中心的 ERC-20 是部分所有权代币，例如NIFTEX 的分片 NFT或Metapurse 的 B20 代币。

    为了深入了解，NIFTEX 是一个平台，可让用户将其 NFT 细分为易于交易且具有流动性的 ERC-20 代币。因此，在 NFT 的集体所有权方面，NIFTEX 是一个很有前途的途径，它已经有大约 20 个 NFT 分片市场可供选择。

    Metapurse 的 B20 代币也采用了类似的想法，因为该集团对 Beeple 20 系列艺术品的所有权进行了分割。随着 Beeple 即将举行的佳士得拍卖会，这可能是一种独特的方式来了解他的作品以及拍卖可能带来的潜在好处。

    如果您认为基础 NFT 的价值会上涨，那么这些部分所有权代币是获得相关风险的一种简单方法。

    向 NFT 用户借出加密货币

    越来越多的 NFT 持有者有兴趣在NFTfi等平台上用 NFT借用加密货币，这是一个 NFT 抵押贷款平台。

    外卖？您可以将加密点对点借给将 NFT 作为抵押品并以此赚取利息的人。如果借款人违约，您将获得基础 NFT 抵押品的奖励。

    来源：nftfi.com/app/lend/assets

    **

    如果这听起来像是您感兴趣的 NFT x DeFi 交叉点，请导航到 NFTfi 的 Lending 仪表板，连接您的浏览器钱包，然后根据您喜欢的任何列出的 NFT 抵押品提议提供贷款。

    购买 NFT 指数基金

    最后，我们开始看到 NFT 指数基金的兴起，这些基金跟踪一揽子类似的 NFT。

    这里的先驱之一是NFTX，它专注于建立跟踪特定 NFT 类别（如 Zombie CryptoPunks 或 Origin Axies）底价的指数基金。

    换句话说，如果您想在您的投资组合中轻松、流动地接触 Zombie CryptoPunks，您可以购买 PUNK-ZOMBIE 代币的一小部分，对于 Origin Axies 而言，ORIGIN-AXIE 也是如此。

    这些 NFTX 指数基金目前在 Sushiswap 上拥有最多的流动性，可用于交易的初始资金包括：

    • PUNK-BASIC（跟踪 CryptoPunks 的底价）

    • PUNK-FEMALE（跟踪女性加密朋克的底价）

    • PUNK-ZOMBIE（追踪 Zombie CryptoPunks 的地板）

    • PUNK-ATTR-4（使用 4 个属性跟踪 CryptoPunks 的底价）

    • PUNK-ATTR-5（使用 5 个属性跟踪 CryptoPunks 的底价）

    • PUNK（由NFTX的各种CryptoPunks指数组成的指数基金）

    总结

    NFT 经济正在以前所未有的方式升温。人们正在意识到这些资产的潜力。因此，任何精明的加密货币投资者都应该考虑如何利用当前的繁荣。

    有很多方法可以获得曝光。你可以直接投资于 NFT，获得以 NFT 为中心的 ERC-20 或 NFT 指数基金，甚至可以将加密货币借给想要使用 NFT 作为抵押品的借款人。但并非没有风险！

    在投资新资产时始终保持谨慎，并通过大量尽职调查和风险考虑来处理您的个人 NFT 策略。

    您在收集工作中投入的心思越多，您建立强大的 NFT 投资组合的机会就越大！

    行动步骤

    • 尝试在OpenSea、Rarible、SuperRare或NiftyGateway上购买 NFT

    • 在 Uniswap 或 Sushiswap 上交易基于 NFT 的代币

    • 调查 NFT 借贷平台，如NFTfi
